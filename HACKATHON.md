# Ovra — AI PM-бот для команды

> **Telegram-бот, который слушает ваши чаты и встречи, создаёт задачи на YouGile и следит за их выполнением — без ручного ввода.**

---

## 🎯 Проблема

Команды тратят часы на перекладывание договорённостей из чата в таск-трекер вручную.
Итоги встреч теряются. Дедлайны забываются. PM превращается в секретаря.

**Ovra это автоматизирует.**

---

## ⚡ End-to-End сценарий

```
Сообщение / голосовой / встреча
         ↓
   AI извлекает задачу + дедлайн + исполнителя
         ↓
   Карточка-подтверждение в Telegram (✅ / ✏️ / ❌)
         ↓
   Задача создана в YouGile (нужная колонка, исполнитель, срок)
         ↓
   Напоминание в личку за 24ч до дедлайна
         ↓
   Реплай «готово» → задача переезжает в Done
         ↓
   Вечерний дайджест: что сделано, что висит
```

## 🎬 Demo сценарий (end-to-end)

1. **Сообщение в чат:** «Антон, сделай лендинг к пятнице»
   → карточка с кнопками ✅/✏️/❌

2. **✅ подтверждение** → задача в YouGile, колонка «В работе», исполнитель Антон, дедлайн пятница

3. **Напоминание** Антону в личку за 24ч

4. **Голосовой ответ** Антона на карточку: «готово» → задача переехала в Done

5. **Вечерний дайджест** показывает: лендинг ✅, что ещё висит

6. **Ссылка Telemost в чате** → бот заходит в встречу → транскрибирует → саммари + новые задачи на доске
